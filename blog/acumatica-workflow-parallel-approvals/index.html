Acumatica Workflow — Parallel Approvals

How to set up parallel approval groups in Acumatica — multiple approvers who must all agree, configuration of approval maps, and the workflow glue that ties them together.

John KihiuFeb 14, 20265 min read

"Three departments must all sign off, then it can proceed" — the parallel approval pattern. Acumatica supports it natively, with some configuration.

Set up the approval map

On the Approval Maps screen (EP205010), define a map that lists the groups, in parallel, with conditions for which group owns a particular document.

Wire the workflow

In the workflow, the transition to Approved requires all groups to have approved. Configure the workflow condition to check the group status, not the overall status.

Don't mix sequential and parallel A workflow with a sequential step that then branches into parallel groups is the most common source of "stuck approvals" tickets. Document the flow on paper first; build the workflow second.
